خصائص البحث

  • البحث عن أي كلمة أو جزء من كلمة (كلما = كَلِمات، كُلَّما)
  • البحث مع التشكيل (كُذِبُوا | كَذَبُوا | كُذِّبُوا | كَذَّبُوا)
  • البحث عن سورة بالاسم أو الرقم (تو: | ٩: = جميع آيات سورة التوبة)؛ (تو:٥ | ٥:٩ = الآية الخامسة من سورة التوبة)؛ ( : = جميع آيات القرآن الكريم)
  • البحث عن الكلمات الموافقة لتشكيل معين (  َ   ّ    ُ    َ = رَبُّكَ)
    • ↓
  • الحركات المستعملة هي: الفتحة والضمة والكسرة وتنويناتها؛ والشدة والسكون؛ والعلامتان الخاصتان (-) و (+)
    • العلامات الخاصة
  • العلامة (-) تستعمل لمطابقة الحروف غير المشكّلة مثل: ال التعريف/الحروف المفردة ( -  -  -  =   الم )؛ وهمزة الوصل (  َ -  ْ    َ   ْ    ِ    ِ  - = وَاسْتَغْفِرِي)؛ وحروف العلة الممدودة ( -   ِ     ِ -  َ =  آمِنِينَ)
  • العلامة (+) تستعمل لمطابقة أيّ وضع (شاملة لجميع الحركات؛ والحالات الخاصة؛ والحركة التي تلي الشدة)
    • استعمال التشكيل
  • علامة الشدة حركة والعلامة الملازمة حركة ثانية (  ُ   ّ     َ =  ثُمَّ)
  • ال الشمسية متبوعة دائما بالشدة (- -   ّ     َ   ْ =  الشَّمْس)؛ وال االقمرية ساكنة اللام (-   ْ    َ    َ =  الْقَمَر)
  • التنوين يسبق الألف (  ُ    ْ     َ -    ً  =  سُلْطَانًا)
